Balanço de estoque

Crie uma contagem, registre as mercadorias e compare o saldo do sistema com a quantidade contada antes de atualizar o estoque.

Como funciona

1. Crie a contagem.
2. Registre os produtos pelo código ou código de barras.
3. Imprima ou exporte o relatório para comparar.
4. Confirme apenas quando tiver certeza para atualizar o estoque.
5. Se criar uma contagem errada, quem criou ou o admin pode remover pela lista.
